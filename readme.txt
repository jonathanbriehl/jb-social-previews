Adds meta tags to WordPress site header to create Twitter summary cards and Facebook previews.

== Description ==

Adds meta data to header to create Twitter cards and Facebook previews. Site wide default settings can be applied as well
as selecting a default image to use when posts or pages do not have a featured image. Additionally, you can override the
default settings on a post by post basis.

= More documentation =

Go to [http://www.jonathanbriehl.com](http://www.jonathanbriehl.com)

== Installation ==

Manual Installation:

	1. Download the plugin, then extract it.
	2. Upload 'jb-social-previews' extracted folder to the '/wp-content/plugins/' directory
	3. Activate the plugin through the 'Plugins' menu in WordPress

You're all set and the plugin is ready to use!
